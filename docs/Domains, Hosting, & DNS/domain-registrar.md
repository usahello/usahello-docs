# Domain Registrar

## USAHello Domain Registration via Namecheap

All USAHello domains are owned and managed through [Namecheap](https://www.namecheap.com).

### Domains

USAHello owns several domains, some actively used and some parked:

- **usahello.org** — primary domain
- **usahello.com** — owned for forwarding purposes
- **therefugeecenter.org** — original organization name, kept to maintain authority and redirect to the main site

There are additional parked domains that were acquired for potential future use.

### USAHello.org Domain Details

| Setting | Value |
|---|---|
| Auto Renew | Enabled |
| Privacy Protection | Enabled (with auto renew) |
| Name Servers | Custom DNS (Cloudflare) |

DNS is routed through Cloudflare — see the Cloudflare documentation for details on DNS management.
